# üìù What is Web Api Template

Web Api Template is a project developed to speed up and simplify project development on the .Net Core Web API.

You can check out how I developed a sample project in this template by visiting the <b>enhancement / product</b> [branch](https://github.com/mwlt68/WebApiTemplate/tree/enhancement/product)

# ü§î What technologies does include ?

* .Net Core Web API project.
* N-Tier architecture based.
* MSsql used.
* Dockerization
* Global Exception Handling
* Caching
* Logging with Serilog
* Repository pattern implementation.
* Advanced Swagger implementation. ([Extension Class](https://github.com/mwlt68/WebApiTemplate/blob/master/src/Core/Extensions/SwaggerExtension.cs))
* JWT implementation for security. ([Extension Class](https://github.com/mwlt68/WebApiTemplate/blob/master/src/Core/Extensions/JwtTokenExtension.cs))

# üöÄ Running the project

1- Installing [docker-compose](https://docs.docker.com/compose/install/).

2- Cloning project

```
git clone https://github.com/mwlt68/WebApiTemplate.git
```
3- Visit root directory of project

4- Compose Up

```
docker-compose up
```
